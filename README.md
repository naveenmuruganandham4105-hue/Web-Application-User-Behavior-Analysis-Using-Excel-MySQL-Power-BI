📊 Web Application User Behavior Analysis Using Excel, MySQL & Power BI
📌 Project Overview

This project focuses on analyzing user behavior data collected from a web application to identify user engagement patterns, conversion trends, and overall application performance. The project demonstrates the complete Data Analytics workflow, including data cleaning, data analysis, and data visualization using industry-standard tools such as Microsoft Excel, MySQL, and Power BI.

The main objective of this project is to transform raw user activity data into meaningful insights that can help businesses improve user experience, increase engagement, and make data-driven decisions.

🎯 Objectives
Clean and preprocess raw user activity data.
Identify meaningful user behavior patterns.
Analyze user engagement and conversion trends.
Visualize insights through interactive dashboards.
Support business decision-making using data analytics.
📂 Dataset Description

The dataset contains 100 records of web application user activity with the following attributes:

Column Name	Description
user_id	Unique identifier for each user
page	Page visited by the user
device	Device type (Desktop, Mobile, Tablet)
session_duration_sec	Time spent on the website
clicks	Number of clicks performed
scroll_percent	Percentage of page scrolled
bounce	Indicates whether the user left immediately (0/1)
conversion	Indicates whether the user completed a goal (0/1)

User Engagement

High Performing Pages
